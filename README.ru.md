# Masterwork Resonance

🌐 Язык: [English](README.md) | [Русский](README.ru.md)

[Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3741570565)

**Masterwork Resonance** - мод для RimWorld, который добавляет редкие случайные бонусы к высококачественному созданному снаряжению.

Когда колонист создаёт оружие, броню или одежду качества **Шедевр** или **Легендарное**, предмет может пробудить случайный эффект **Резонанса**. Бонус сохраняется прямо на предмете и отображается в панели осмотра предмета.

## Возможности

* Случайные бонусы резонанса для созданных шедевров и легендарных предметов
* Работает с оружием ближнего боя, дальнобойным оружием, бронёй и одеждой
* 40% шанс пробуждения для шедевров
* 80% шанс пробуждения для легендарных предметов
* Цветовое отображение силы броска резонанса
* Бонус настроения отображается во вкладке потребностей пешки
* Английская и русская локализация
* Поддержка RimWorld 1.5 / 1.6
* Детерминированная генерация при крафте, дружелюбная к Multiplayer

## Шансы пробуждения

| Качество    | Шанс резонанса |
| ----------- | -------------: |
| Шедевр      |            40% |
| Легендарное |            80% |

Если резонанс не пробудился, предмет остаётся обычным шедевром или легендарным предметом.

Уже существующие предметы не получают резонанс задним числом. Резонанс может пробудиться только у новых созданных предметов.

## Возможные эффекты резонанса

Оружие может получить такие бонусы:

* Бонус урона в ближнем или дальнем бою
* Снижение перезарядки ближнего боя
* Снижение перезарядки дальнобойного оружия
* Снижение времени прицеливания
* Бонус точности ближнего боя
* Бонус точности стрельбы
* Бонус уклонения в ближнем бою
* Бонус настроения при экипировке

Броня и одежда могут получить такие бонусы:

* Бонус скорости передвижения
* Улучшение брони
* Увеличение прочности предмета
* Снижение получаемого урона при ношении
* Бонус уклонения в ближнем бою
* Бонус настроения при ношении

## Цвета силы броска

Каждый резонанс бросается в пределах своего диапазона значений.

| Цвет      | Значение       |
| --------- | -------------- |
| Бронзовый | Низкий бросок  |
| Оранжевый | Средний бросок |
| Зелёный   | Высокий бросок |

Даже низкий бросок остаётся положительным бонусом. Цвет показывает только то, насколько сильным получился бонус относительно максимального значения для этого резонанса.

## Резонанс настроения

Предметы с бонусом настроения добавляют видимую строку во вкладке потребностей пешки:

**Резонирующее снаряжение**

Бонус применяется только пока резонирующий предмет экипирован или надет.

## Совместимость

Мод рассчитан на совместимость с другими модами которые  создают оружие, броню и одежду, если предмет:

* использует систему качества RimWorld
* создаётся качеством Шедевр или Легендарное
* распознаётся игрой как подходящее оружие или одежда/броня

## Multiplayer

Генерация резонанса использует детерминированные броски при крафте.

Базовые тесты крафта в Multiplayer были выполнены без рассинхронов, а значения созданного резонанса совпадали между игроками.

* Русский

Требуется [Harmony](https://steamcommunity.com/sharedfiles/filedetails/?id=2009463077).

## Сборка из исходников

Проект собирается как C# мод для RimWorld.

Нужны ссылки на:

* `Assembly-CSharp.dll`
* `UnityEngine.CoreModule.dll`
* `UnityEngine.IMGUIModule.dll`
* `UnityEngine.TextRenderingModule.dll`
* `0Harmony.dll`

После сборки готовую DLL нужно положить в:

```text
Assemblies/
```
